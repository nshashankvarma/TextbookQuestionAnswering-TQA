Datasets:
1. TQA Dataset - https://allenai.org/data/tqa
2. Flickr Dataset - https://www.kaggle.com/datasets/adityajn105/flickr8k

##Caption Generation
This code generates captions for all the images taken from the texbooks and appends them to the text to get contexts from both of them.

##CNN-TQA
This code trains a BERT model and generates answers for the textbook questions using context from both images and text.
